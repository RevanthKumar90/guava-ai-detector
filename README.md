# 🌿 Guava AI Detector

An AI-powered Android application for **Guava Leaf Disease Detection** using **TensorFlow Lite**. The application enables users to capture or upload images of guava leaves and instantly predicts the disease using an optimized on-device deep learning model.

---

## ✨ Features

- 📷 Capture leaf images using the camera
- 🖼️ Select images from the device gallery
- 🤖 AI-powered disease prediction
- ⚡ Fast on-device inference with TensorFlow Lite
- 📱 Simple and user-friendly Android interface
- 🌱 Supports early disease identification

---

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** Android Studio
- **Machine Learning:** TensorFlow Lite
- **UI:** XML
- **Build Tool:** Gradle

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/RevanthKumar90/guava-ai-detector.git
```

### 2. Open the project

Open the project in **Android Studio**.

### 3. Sync Gradle

Allow Android Studio to download the required dependencies.

### 4. Run the application

Connect an Android device or start an emulator and run the project.
